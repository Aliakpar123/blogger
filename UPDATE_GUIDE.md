# Как обновить код на GitHub

Каждый раз, когда вы что-то меняете в файлах (`index.html`, `script.js` и т.д.), нужно "отправить" эти изменения на GitHub, чтобы сайт обновился.

## Пошаговая инструкция (Терминал)

1. **Откройте Терминал**.
2. **Перейдите в папку проекта** (если еще не там):
   ```zsh
   cd "/Users/aliakbar/Вишт лист для блоггеров "
   ```
3. **Выполните 3 команды** (можно копировать все сразу):

   ```zsh
   git add .
   git commit -m "Обновление: Kaspi Pay и исправления"
   git push
   ```

### Что делают эти команды?
1. `git add .` — собирает все измененные файлы.
2. `git commit -m "..."` — "запаковывает" их с комментарием (описанием, что изменилось).
3. `git push` — отправляет "пакет" на GitHub.

---

## Если вы используете сайт GitHub (без терминала)
1. Зайдите в ваш репозиторий на [github.com](https://github.com).
2. Нажмите кнопку **Add file** -> **Upload files**.
3. Перетащите обновленные файлы (`script.js`, `index.html`, `style.css`) в окно.
4. Внизу нажмите зеленую кнопку **Commit changes**.

*После этого подождите 1-2 минуты, и изменения появятся на вашем сайте.*
