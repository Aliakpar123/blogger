# Инструкция: Как запустить сайт с Mac (macOS)

Эта инструкция поможет вам опубликовать ваш **Wish List** в интернете, используя ваш MacBook.

## Способ 1: Самый простой (Vercel)
Этот способ не требует работы с кодом, просто перетаскивание файлов.

1. Откройте папку с проектом в **Finder**:
   - Путь: `/Users/aliakbar/Вишт лист для блоггеров`
2. Откройте сайт [vercel.com](https://vercel.com) в браузере (Safari/Chrome).
3. Зарегистрируйтесь (можно через GitHub или почту).
4. Нажмите кнопку **"Add New..."** -> **"Project"**.
5. НЕ выбирайте репозиторий, а найдите кнопку **"Upload"** или поле **"Import Third-Party Git Repository"** -> но проще всего:
   - Если у вас установлен **Vercel CLI** (через терминал), это быстрее.
   - Если нет, то **GitHub** (Способ 2) надежнее.

---

## Способ 2: Через GitHub (Рекомендуемый)

### Шаг А: Подготовка на Mac
1. Откройте приложение **Терминал** (Command + Пробел -> введите "Terminal").
2. Введите команду, чтобы перейти в папку проекта:
   ```zsh
   cd "/Users/aliakbar/Вишт лист для блоггеров "
   ```
3. Инициализируйте Git (если еще не сделали):
   ```zsh
   git init
   git add .
   git commit -m "Первая версия Wish List"
   ```

### Шаг Б: Загрузка на GitHub
1. Создайте **новый репозиторий** на сайте [github.com](https://github.com/new).
   - Назовите его, например: `wish-list-app`.
   - Не ставьте галочки (README, gitignore).
2. Скопируйте **код для загрузки** (он появится на GitHub после создания, начинается с `git remote add origin...`).
3. Вставьте этот код в ваш Терминал и нажмите Enter.
4. Введите команду для отправки:
   ```zsh
   git push -u origin main
   ```

### Шаг В: Включение сайта
1. На странице репозитория на GitHub зайдите в **Settings** (Настройки).
2. Слева выберите раздел **Pages**.
3. В пункте **Branch** выберите `main` и нажмите **Save**.
4. Подождите 1-2 минуты, обновите страницу. Сверху появится ссылка на ваш сайт!

---

## Способ 3: Для продвинутых (Vercel CLI через Терминал)
Если у вас установлен Node.js:

1. Откройте Терминал.
2. Перейдите в папку:
   ```zsh
   cd "/Users/aliakbar/Вишт лист для блоггеров "
   ```
3. Запустите deploy:
   ```zsh
   npx vercel
   ```
4. Нажимайте `Enter` (Yes) на все вопросы.
5. В конце вы получите готовую ссылку (Production: https://...).
